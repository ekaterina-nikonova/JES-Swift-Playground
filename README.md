# Japanese Exercise System (JES) – Swift Playground

This is an early prototype of the [iPad application](https://github.com/ekaterina-nikonova/JES.git) that serves as a GUI for the [JES project](https://github.com/ekaterina-nikonova/3070-final-project).

This prototype uses "canned" server responses and demonstrates the feasibility of collecting and storing Japanese handwriting and voice recordings on an iPad. 
For complete functionality including invocation of LLMs on a local web-server, use the complete [iPad application](https://github.com/ekaterina-nikonova/JES.git).
